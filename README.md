# Downloading Instructions
Download and unzip game.zip. All dependencies required to run the game are inside. 
# Controls
  - W - Move Forward
  - S - Move Backward
  - A - Move Left
  - D - Move Right
  - Spacebar - Jump
  - Left-Shift - Run (When combined w/ any move key)
  - P - Pause
